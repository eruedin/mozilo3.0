Sprache

Als erstes kann die gewünschte Sprache eingestellt werden. moziloCMS kommt bereits mit einer Vielzahl an Übersetzungen.

Die gewählte Sprache wird zunächst für die Webseite und den Admin-Bereich übernommen. Nach der Installation kann im Admin-Bereich, im Tab "Einstellungen", die Sprache der Webseite und im Tab "Admin" die der Administrationsoberfläche geändert werden.

Datei- und Zugriffsrechte

Es wird versucht die Server-Dateirechte zu ermitteln und zu setzen, die für das einwandfreie Funktionieren von moziloCMS nötig sind. Sollte dies nicht vollständig gelingen, können sie im Admin-Bereich, im Tab "Admin", nachträglich gesetzt werden.

Falls der Server moziloCMS daran hindert Dateien anzulegen, müssen die dafür nötigen Dateirechte zuerst per FTP, mit dem chmod-Befehl gesetzt werden, bevor die Installation fortgesetzt werden kann.

Serverumgebung

Der Webserver wird auf die für moziloCMS notwendigen Grundvoraussetzungen hin geprüft. Nicht hervorgehobene Funktionen sind optional, d.h. moziloCMS funktioniert grundlegend. Du musst aber unter Umständen auf die entsprechende Funktion verzichten.

Schöne URLs

moziloCMS versucht, auf dem Server das mod_rewrite-Modul zu aktivieren. Dadurch werden besser lesbare und suchmaschinenfreundlichere URLs erzeugt.

Ist mod_rewrite verfügbar, kann es später im Admin-Bereich, im Tab "Einstellungen", nach Wunsch deaktiviert oder aktiviert werden.

Sofern mod_rewrite verfügbar ist, wird eine .htaccess Datei angelegt. "mod_rewrite ist verfügbar und wurde durch Anlegen einer .htaccess-Datei aktiviert." Anderenfalls erscheint folgende Meldung "mod_rewrite ist nicht verfügbar."
